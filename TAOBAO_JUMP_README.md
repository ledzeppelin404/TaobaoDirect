# 淘口令跳转功能说明

## 功能介绍
在微信聊天中长按消息，菜单中会出现"跳转淘宝"选项，点击后自动复制消息内容并打开淘宝App。

## 使用方法

### 1. 启用功能
1. 打开微信，进入"设置" → "插件" → "微信设置"
2. 找到"功能增强"区域，点击"淘口令跳转"
3. 打开"启用淘口令跳转"开关
4. 重启微信使设置生效

### 2. 使用功能
1. 在聊天界面，当有人发送淘口令或商品链接时
2. **长按该消息**
3. 在弹出的菜单中选择"**跳转淘宝**"
4. 系统会自动复制消息内容并打开淘宝App

## 适用场景
- 淘口令（￥xxxxx￥）
- 淘宝商品链接
- 天猫商品链接
- 其他需要在淘宝中打开的内容

## 注意事项
- 需要手机已安装淘宝App
- 如果未安装淘宝，会提示"无法打开淘宝"
- 功能开启后需要重启微信才能生效
